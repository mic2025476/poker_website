{% trans "Overview" %}

{% blocktrans %} This policy explains how cancellations and refunds work for bookings and event-related services offered by MGEN Poker Lounge. Our services include the rental of poker tables and related event equipment and may include beverage flat-rate packages depending on the booking. {% endblocktrans %}

{% trans "Section 2" %}

{% trans "No Right of Withdrawal for Event Services" %}

{% blocktrans %} For contracts related to leisure activities with a specific date or time period, there is generally no right of withdrawal under German law (e.g., § 312g Abs. 2 Nr. 9 BGB). This typically applies to reservations and event bookings for a defined date. {% endblocktrans %}

{% trans "Section 3" %}

{% trans "Cancellations by the Customer" %}

{% blocktrans %} Customers may request a cancellation up to 7 days before the scheduled event date (unless a different deadline is agreed in writing). If cancelled in time, we may refund payments in full or in part depending on the booking and costs already incurred. {% endblocktrans %}

  • {% trans "Late cancellations (less than 7 days) may not be refundable." %}
  • {% trans "No-shows are generally not eligible for a refund." %}
  • {% trans "Custom or corporate events may have separate written cancellation terms." %}
{% trans "Section 4" %}

{% trans "Cancellations by Us" %}

{% blocktrans %} If we must cancel an event for reasons within our responsibility, we will refund payments already made for the affected booking. Refunds are issued to the original payment method used. {% endblocktrans %}

{% trans "Section 5" %}

{% trans "Refund Processing" %}

{% blocktrans %} Approved refunds are processed via the original payment method (e.g., PayPal) and are typically completed within 5–10 business days, depending on the payment provider and bank processing times. {% endblocktrans %}

{% trans "Section 6" %}

{% trans "Contact" %}

{% blocktrans %} If you have questions about cancellations or refunds, please contact us using the details in our Imprint / Impressum. {% endblocktrans %}
